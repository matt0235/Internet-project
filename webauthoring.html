WEB AUTHORING

Web authoring means making stuff for websites. It's about creating and designing web pages using tools and languages like HTML, CSS, and JavaScript. Web authors, also known as web developers or designers, use these tools to organize content and make it look good.

Sublime text

Sublime Text is a popular and lightweight text editor widely favored by developers for its speed and simplicity. Known for its clean interface and powerful features, Sublime Text provides a versatile environment for writing code in various programming languages.

Visual Studio Code

Visual Studio Code, is a popular free tool made by Microsoft for writing code. It's widely used by developers because it's easy to use and has lots of helpful features.
You can write code in many different languages, and it helps you with things like finding mistakes, debugging, and keeping track of changes.
